mORK QR Code Generator

You can search for a player and have the QR code generated for the player selected. Some hints for searching that apply to the ORK itself, mORK, and this page. You can prefix a search with a Kingdom abreviation to narrow a player search to just that kingdom. So if you searched for "nb: kismet" only the Nine Blades would return results. You can also print this page once the QRCode is generated and just the relevant information will be included.
Chrome is the only browser currently working for the copy image to clipboard.
