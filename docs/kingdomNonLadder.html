Kingdom NON-LADDER Awards

This report will show all players who are at or above the chosen level in each of the non-ladder awards. This document relies on award entries in the ORK. If an entry doesn't seem accurate, check the player on the ORK. Each player has a link to the ORK proper. By default the report only includes players who have reached level 7 or higher.

You can print the results from the browser after the query has been completed.
You can copy each individual award list to the clipboard for pasting into a spreadsheet.
