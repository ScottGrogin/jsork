jsork - JS Amtgard ORK APIs

Javascript Amtgard ORK APIs by Kismet of Felfrost

jsork is a collection of Javascript APIs that communicate to the Amtgard ORK. These APIs can be used to query many things in the ORK and can handle login authentication for manipulation or creation of entities within the ORK.

The jsork APIs are what power the mobile application mORK available on the iOS and Android App stores.

The APIs have been grown a bit organically by the requirements of mORK and other user build web applications needing to talk to the ORK.

If you do not call the login API then there's no need to worry that any calls will modify or remove data from the ORK itself. If you do call the login APIs then subsequent calls will be made with whatever permission you've been granted on the ORK. Park and Kingdom officers will be able to perform all create, modify and delete options across the entities within their particular level. ORK Admins will be able to do the same but across the entire set of ORK entities.
