jsork - JS Amtgard ORK APIs

jsork is a collection of Javascript APIs that communicate to the Amtgard ORK. These APIs can be used to query many things in the ORK and can handle login authentication for manipulation or creation of entities within the ORK. The APIs have been grown a bit organically by the requirements of mORK.
